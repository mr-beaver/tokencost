Model & Effort — what to use and when
ModelEffortThinking budget Cost (input/output)Best forExample tasks
Haiku 4.5 — none $1 / $5 Ultra-fast simple tasks Grep, file lookup, "what is X?", yes/no questions, linting
Sonnet 4.6 standard none $3 / $15 Everyday coding Bug fixes, refactoring, writing functions, code review, tests
Sonnet 4.6 medium ~4k tokens $3 / $15 + think Tricky bugs & design Multi-file changes, "why does this fail?", API design, SQL queries
Sonnet 4.6 high ~8–12k tokens $3 / $15 + think Complex features Architecture decisions, hard performance issues, security audit
Opus 4.7 medium ~4k tokens $5 / $25 + think Critical decisions System design, production incidents, compliance review, tech lead tasks
Opus 4.7 xhigh >12k tokens $5 / $25 + think Research & hard math Novel algorithms, deep optimization, open-ended research, PhD-level problems
💡 Tip: thinking tokens bill at output rate. Sonnet high = $15/MTok thinking + $15/MTok output. Set effortLevel: "medium" in ~/.claude/settings.json as default — use high/xhigh only when you need deep reasoning.
